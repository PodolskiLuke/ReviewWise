# ReviewWise Frontend (Angular)

## Overview
This is the Angular frontend for ReviewWise. It provides a dashboard for managing repositories, viewing pull requests, and displaying AI review results.

## Tech Stack
- Angular
- SCSS
- REST API integration with backend
- OAuth login (GitHub/GitLab)

## Setup
1. Install [Node.js](https://nodejs.org/) and [Angular CLI](https://angular.dev/tools/cli)
2. Navigate to `/frontend` and run `npm install`
3. Run `ng serve` to start the development server

## Development
- Update environment variables in `src/environments/` as needed
- Use Angular CLI for scaffolding components/services
- Connect to backend API via environment config
