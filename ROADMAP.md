# ReviewWise – Roadmap

## MVP
- [ ] User authentication (GitHub/GitLab OAuth)
- [ ] Connect repositories
- [ ] Fetch and display pull requests
- [ ] Run AI review on PRs (suggestions, comments)
- [ ] Dashboard for PRs and review results

## Next Steps
- [ ] Customizable code review rules
- [ ] Team management (invite/manage users)
- [ ] Notifications (email, in-app)
- [ ] Security checks and code style enforcement

## Advanced
- [ ] Inline code suggestions
- [ ] Learn from team feedback
- [ ] Analytics dashboard
- [ ] Support for multiple platforms (GitHub, GitLab, Bitbucket)
